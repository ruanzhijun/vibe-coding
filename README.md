## 使用方式

可以把这个仓库当作个人或团队的 vibe coding 知识库：

- 文档层：沉淀最佳实践和配置说明
- 模板层：沉淀可直接复用的配置文件
- 项目层：为不同仓库复制或裁剪一套合适的 Codex 配置
